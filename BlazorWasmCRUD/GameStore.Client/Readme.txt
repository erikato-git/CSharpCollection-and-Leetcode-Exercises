Min oplevelse af Blazor:
Nogle af tingene mht. at forbinde C# i HTML sammen med metoder skrevet i selve razor-klasserne giver fint mening. 
Dog er det en meget dårlig developer oplevelse, jeg har af frameworket, da der opstår nogle mystiske fejl undervejs som VS ikke gør én opmærksom på fx markere med rød, hvord der er syntax-fejl.
Derudover er load-tiden al for langsom, når man kører projektet. 

Konklusion:
Jeg holder mig til ASP.NET Core Web API med React som front-end eller ASP.NET MVC. Blazor er der ikke endnu til, at jeg vil udvikle i det.
